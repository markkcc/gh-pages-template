TMUX

tmux is a terminal multiplexer: A program that lets you split your terminal into multiple windows. You can dispaly panes side by side and work on several programs at the sime time, switching between them quickly using keybindings. tmux can be started with the following commands: 
tmux new -s basic -n window_name # Create a new session called basic and name the first window
tmux new -s detached_session_name -d # Creates a session in the background
tmux ls # List existing sessions already created
tmux attach # Attach to the only existing session
tmux attach -t session_name # Attach to a specific session
tmux kill-session -t session_name # Kills a specific session
The prefix command is used to differentiate between commands going to the running program, and commands intended to go to tmux itself. By default, the prefix command is: 
Ctrl-b
Deault keybindings: 
Ctrl-b + t # Show the current time
Ctrl-b + d # Detach from a screen
Ctrl-b + c # Create a new window
Ctrl-b + , # Rename current window
Ctrl-b + n/p/0/1/2/... # Next/Previous window, or go to window number directly
Ctrl-b + w # Show all windows
Ctrl-b + % # Split window vertically |
Ctrl-b + ” # Split window horizontally -
Ctrl-b + o/up/down/left/right # Cycle through panes
Ctrl-b + Space # Cycle through different layouts
Ctrl-b + x # Kill pane or window
Ctrl-b + ? # List all default keybindings

NMAP

Scan the fuck out of a single port: 
nmap -vvv -A --reason --script="+(safe or default) and not broadcast" -p 4567 192.168.1.1

GENERATE RANDOM STRING

cat /dev/urandom | env LC_CTYPE=C tr -dc 'a-zA-Z0-9!@#$%^&*()_+-=[]{}\|;:",./<>?`~' | fold -w 32 | head -n 1

SCP

To copy a file from B to A while logged into B: 
scp /path/to/file username@a:/path/to/destination
To copy a file from B to A while logged into A: 
scp username@b:/path/to/file /path/to/destination

CHECK FILE/FOLDER SIZE

du -sh
du -sh /folder

FIND BY FILENAME

Find . -name test.java

REGEX

Remove all characters starting from "- " till the end of the line:
(?s) -.*?$

ZIP

Create an encrypted zip of a file or folder. You will be prompted for a password.
zip -e out.zip filename
zip -er out.zip foldername
